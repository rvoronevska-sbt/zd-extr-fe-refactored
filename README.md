# 🧮 Zendesk Extractor Frontend Project

The Zendesk Extractor Frontend Project is based on the [Sakai PrimeVue Theme](https://github.com/primefaces/sakai-vue) with [Live View](https://sakai.primevue.org/).

Sakai is an application template for Vue based on the [create-vue](https://github.com/vuejs/create-vue), 
the recommended way to start a Vite-powered Vue projects.
